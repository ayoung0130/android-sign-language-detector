## 의료시설용 실시간 수어 통역 애플리케이션

- **MediaPipe Hands, Pose 모델을 활용해 사용자의 동작 데이터를 수집**
- **시계열 데이터로 가공하여 TFLite 모델에 입력**
- **모델이 예측한 결과를 LLM 모델을 활용해 자연스러운 문장으로 변환**
- **변환된 문장과 TTS 음성을 함께 출력**
